AdventShare is built using the following technologies:

- **Server**: [Hetzner](https://www.hetzner.com/)  
- **Database**: [Supabase](https://supabase.com/)
- **Auth Provider**: [Clerk](https://clerk.dev/)
- **JavaScript Runtime**: [Bun](https://bun.sh/)
- **Programming Language**: TypeScript (frontend & backend)
- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Toasts**: [sonner](https://sonner.emilkowal.ski/)
- **Input Validation**: [zod](https://zod.dev/)
- **Date Handling**: [date-fns](https://date-fns.org/)
- **Secure Server Actions**: [next-safe-action](https://github.com/TheEdoRan/next-safe-action)
- **Email Provider**: [Resend](https://resend.com/)
